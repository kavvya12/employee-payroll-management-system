# Employee Payroll Management System

This is a Java-based project that manages employee data, tracks attendance, and calculates salary using Object-Oriented Programming concepts.

## Features
- Add employee details (ID, name, salary per day)
- Track attendance (number of days present)
- Calculate salary based on attendance
- Display employee details

## Tech Stack
- Java
- OOP Concepts

## How it Works
The system takes employee input such as ID, name, salary per day, and attendance. It then calculates the total salary based on the number of working days.
